## 项目启动阶段

### L001 · PRD 先行的重要性

**发现：** 本项目先完成完整 PRD + 开发实施文档再开始编码，技术决策有理有据

**教训：** 需求文档不是"写完就丢"的产物 — Dev Spec 中的 API 设计、数据库 schema、组件树结构直接指导编码，减少架构返工
